# 搜索与筛选页面

## 概述

搜索页面提供多维度检索功能，让用户通过日期范围、任务状态、任务标签、优先级等条件组合查找求职任务。搜索结果按截止时间排序，支持在时间视图中定位任务位置。搜索可从任意页面通过顶部导航栏的搜索按钮或快捷键 `Ctrl/Cmd + K` 触发。

## 入口方式

搜索有两种入口形态：

1. **搜索浮层**（快捷搜索）：从顶部导航栏展开，覆盖在当前页面上方，适合快速查找和定位
2. **搜索完整页面**：当用户需要更复杂的筛选组合时，跳转到的独立页面

### 搜索浮层

- **触发**：点击导航栏搜索按钮 / 快捷键 `Ctrl/Cmd + K`
- **位置**：顶部导航栏下方，居中，宽度 640px，最大高度 80vh
- **背景**：`#FFFFFF`，圆角 xl，阴影 `shadow-xl`
- **动画**：从顶部淡入+轻微下移（opacity 0→1 + translateY -8px→0，200ms）
- **结构**：
  - 搜索输入框（顶部，height 48px，大字号 H2，placeholder "搜索公司、岗位、标签..."）
  - 快捷筛选行（输入框下方，标签式单选：全部/今天/本周/面试中/即将截止）
  - 搜索结果区（下方，最大高度 60vh，纵向滚动）
  - 底部提示行："按 Enter 查看完整搜索结果" / "按 Esc 关闭"
- **结果展示**：与完整页面的结果列表样式相同（紧凑版），每条结果显示公司名+岗位+状态+日期
- **交互**：点击结果 → 关闭浮层 → 在当前视图中高亮定位该任务（脉冲动画）→ 打开任务详情抽屉

## 页面头部

- **搜索输入框**：占据页面顶部区域，height 56px，H1 24px 字号
  - 左侧：🔍 图标（24px/Text Tertiary）
  - 中间：输入文字
  - 右侧：清除按钮（×，有输入时显示）+ 快捷键提示 "Esc"（Caption/Text Tertiary）
  - 背景：透明，底部边框 2px `#E8EBF0`，focus 时边框变为 Primary Blue
  - placeholder："搜索公司名、岗位名称、笔记内容、标签..."

- **结果统计**：输入框下方，Caption/Text Secondary
  - 无搜索词："输入关键词开始搜索"
  - 有结果："找到 N 个结果"（N 用 Primary Blue 高亮）
  - 无结果："未找到相关结果，试试其他关键词？"

## 筛选面板

左侧固定面板，宽度 240px，padding 16px，纵向滚动。

### 筛选面板结构

每个筛选分类为一个可折叠的 section：

**日期范围**
- Section 标题："📅 日期范围"（H3 15px/600）+ 折叠箭头
- 选项（单选）：
  - "全部时间"（默认选中）
  - "今天"
  - "本周"
  - "本月"
  - "未来7天"
  - "未来30天"
  - "已过期"（显示截止日期已过的任务）
  - "自定义范围" → 点击展开日期范围选择器（开始日期-结束日期）
- 选中项：文字 Primary Blue + 左侧圆点指示

**任务状态**
- Section 标题："🏷 状态"
- 选项（多选，复选框）：
  - 准备中（灰色圆点）
  - 已投递（蓝色圆点）
  - 笔试中（橙色圆点）
  - 面试中（青色圆点）
  - 等结果（紫色圆点）
  - 已Offer（绿色圆点）
  - 已拒绝（红色圆点）
- 每个复选框左侧带对应状态色圆点（8px）
- 全选/清除链接（section 标题右侧）

**任务标签**
- Section 标题："🏷 标签"
- 选项（多选，复选框）：
  - 用户自定义的所有标签，每项带彩色圆点
- 底部："+ 管理标签"链接（打开标签管理弹窗）

**优先级**
- Section 标题："🔥 优先级"
- 选项（多选，复选框）：
  - 🔴 高
  - 🟡 中
  - 🟢 低
  - 用户自定义优先级（带对应颜色）

**排序方式**
- Section 标题："⇅ 排序"
- 选项（单选，互斥）：
  - "按截止时间升序"（默认）
  - "按截止时间降序"
  - "按优先级高→低"
  - "按优先级低→高"
  - "按创建时间最新"
  - "按公司名称A-Z"
  - "按状态顺序"
- 说明：当用户选择"按状态顺序"时，结果按准备中→已投递→笔试中→面试中→等结果→已Offer→已拒绝的顺序排列

### 筛选面板交互

- **实时筛选**：任何筛选条件变更后，结果列表实时更新（debounce 200ms）
- **激活标记**：有选中项的 section 标题右侧显示激活数量小红点
- **折叠/展开**：点击 section 标题可折叠/展开，折叠时只显示标题和激活标记
- **重置**：面板顶部有一个"清除全部筛选"链接（Text Secondary），点击后所有条件恢复默认

## 搜索结果列表

右侧主区域，纵向滚动列表。

### 结果行（Result Row）

每条结果一个行项目，高度 72px，padding 12px 20px，底部分隔线 1px `#F1F3F5`。

- **左侧指示区**（4px宽）：
  - 状态色竖条（整行高度，圆角 full）
- **内容区**（flex 1）：
  - **第一行**：
    - 状态徽章（小尺寸）
    - 公司名称（H3 15px/600/Text Primary）
    - 岗位名称（14px/Text Secondary）— 搜索匹配的关键词在该行中高亮显示（Primary Blue 背景 20%）
  - **第二行**：
    - 标签列表（最多2个，Caption，超出"+N"）
    - 截止时间（Caption，如"截止：3月15日"）
      - > 7天：Text Tertiary
      - 3-7天：Warning Orange
      - < 3天：Danger Red
    - 优先级图标（🔴🟡）
- **右侧操作区**（hover显示）：
  - "定位"按钮（地图定位图标，Caption/Primary Blue）— 点击后在日历视图中定位该任务
  - "编辑"按钮（铅笔图标）
- **Hover**：背景 Surface Hover

### 匹配高亮规则

搜索关键词在以下字段中匹配：
- 公司名称（最高优先级，匹配时公司名高亮）
- 岗位名称（高优先级）
- 标签名称（中优先级）
- 笔记内容（中优先级，匹配时显示笔记片段预览）
- 附件文件名（低优先级）

匹配到的关键词用 Primary Blue 背景 20% + 文字 Primary Blue 高亮显示。

### 空状态

- 图标：🔍 搜索图标放大版（48px/Text Tertiary）
- 标题："未找到相关结果"（H2 18px/600）
- 描述："试试调整筛选条件，或使用更宽泛的关键词"（14px/Text Secondary）
- 操作："清除所有筛选"按钮

### 加载状态

- 骨架屏：5行灰色骨架矩形（每行模仿结果行布局）
- 脉冲动画（shimmer，背景渐变从左到右扫过）

## 定位到时间视图

点击结果行的"定位"按钮后：
1. 关闭搜索页面/浮层
2. 跳转到时间视图（月历/周历/日，优先使用用户当前所在的视图）
3. 目标日期格子/区域高亮显示（脉冲动画：边框 2px Primary Blue，scale 1→1.02→1，持续 800ms）
4. 目标任务卡片高亮（背景变为 Primary Light，持续 2s 后恢复正常）
5. 如果任务不在当前可视范围内，自动滚动到该任务所在位置

## 备注

### 搜索性能优化

- 搜索输入使用 debounce（200ms），避免频繁请求
- 最少输入2个字符才触发搜索（防止单字母匹配过多结果）
- 服务端搜索+本地缓存，已加载的数据在本地做二次筛选
- 结果分页：首次加载20条，滚动到底部时加载更多

### 搜索快捷键设计

- `Ctrl/Cmd + K`：打开搜索浮层
- `Ctrl/Cmd + Shift + F`：打开完整搜索页面
- `↑↓`：在搜索结果中上下移动（浮层和完整页面通用）
- `Enter`：打开选中的结果
- `Esc`：关闭搜索浮层/返回上一页
- `/`：焦点直接移到搜索输入框（当在日历视图时）

### 高级搜索语法

支持简单的搜索语法让重度用户更快找到内容：

| 语法 | 说明 | 示例 |
|------|------|------|
| `status:面试中` | 按状态筛选 | `status:面试中 产品经理` |
| `tag:互联网大厂` | 按标签筛选 | `tag:互联网大厂` |
| `priority:高` | 按优先级筛选 | `priority:高` |
| `before:2025-03-15` | 截止日期之前 | `before:2025-03-15` |
| `after:2025-03-01` | 截止日期之后 | `after:2025-03-01` |
| `company:字节` | 按公司名筛选 | `company:字节` |

输入这些语法时，搜索框下方实时显示解析后的筛选条件预览（如"状态：面试中 · 关键词：产品经理"），帮助用户确认搜索意图。

### 保存搜索

用户可以将常用的筛选组合保存为"快捷筛选"：
- 筛选面板底部有"保存当前筛选"按钮
- 点击后弹出输入框命名该筛选（如"本周面试"）
- 保存后出现在侧边栏"标签区"下方的新 section"快捷筛选"
- 点击快捷筛选 = 直接应用该筛选组合并跳转到搜索页
- 快捷筛选可编辑名称和删除

### 筛选面板在浮层中的简化

搜索浮层中不显示完整筛选面板，而是：
- 搜索框下方显示一排快捷筛选标签（全部/今天/本周/面试中/即将截止/已Offer）
- 点击"更多筛选"展开迷你筛选面板（浮层右侧滑出，宽度 200px）
- 迷你面板只显示最常用的筛选：状态 + 标签 + 排序
