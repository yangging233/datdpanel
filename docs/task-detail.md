# 任务详情抽屉

## 概述

任务详情抽屉是从页面右侧滑出的覆盖面板，用于查看、编辑和管理工作任务的所有信息。抽屉保持用户当前所在的日历/看板视图在背景中可见，确保用户不会"迷路"，编辑后可立即回到上下文继续操作。

### 抽屉规格

- **宽度**：480px（固定），视口 < 768px 时变为全屏（100vw）
- **高度**：100vh
- **位置**：固定右侧（position: fixed; right: 0; top: 0）
- **背景**：`#FFFFFF`
- **阴影**：`shadow-xl`（左侧强烈阴影，与背景页面产生深度分离）
- **动画**：从右侧滑入（translateX 100% → 0，350ms cubic-bezier(0.32, 0.72, 0, 1)）
- **关闭方式**：点击遮罩区 / 按 Esc / 点击关闭按钮
- **关闭动画**：translateX 0 → 100%（250ms），遮罩 opacity 淡出

### 遮罩层

- 背景：`rgba(0, 0, 0, 0.2)`
- 点击可关闭抽屉
- 动画：opacity 0 → 1（200ms）

## 头部区域（Header）

高度 56px，padding 16px，底部分隔线 1px `#E8EBF0`，flex 布局。

- **左侧**：关闭按钮（× 图标，24px，hover 背景 Surface Hover，圆角 full）
- **中部**：标题"任务详情"（H3 15px/600/Text Secondary），编辑模式下变为"编辑任务"
- **右侧**：操作按钮组
  - 编辑按钮（铅笔图标，仅查看模式显示）
  - 更多菜单（⋮ 图标，下拉：复制任务/迁移到速记/删除）

## 内容区域

内容区纵向滚动（overflow-y: auto），padding 20px 24px，各区域之间用 1px `#F1F3F5` 分隔线隔开。

### 基本信息区

**查看模式**：
- **公司名**：H1 24px/700/Text Primary（最大的视觉焦点）
- **岗位名**：H2 18px/500/Text Secondary，公司名下方 4px
- **状态与优先级行**：margin-top 12px，flex 排列
  - 状态徽章（使用全局 StatusBadge 组件，大尺寸版）
  - 优先级徽章（🔴🟡🟢 + 自定义名称，如"高优先级"、"紧急"）
  - 标签列表（彩色圆点+名称，横向排列，间距 8px，可点击跳转筛选）
- **期望值**：如果有填写，显示"期望度：⭐⭐⭐☆☆"（5星制，Caption）

**编辑模式**：
- 公司名 → 输入框（height 44px，H1字号，placeholder "公司名称，如字节跳动"）
- 岗位名 → 输入框（height 36px，H2字号，placeholder "岗位名称，如产品经理"）
- 状态 → 下拉选择（7个状态选项，每项带状态色圆点+名称）
- 优先级 → 下拉选择（预设+用户自定义优先级列表，每项带颜色竖条）
- 标签 → 多选标签组件（可搜索，可新建标签，带彩色圆点）
- 期望值 → 5星选择（点击设置1-5星，hover时预览）

### 时间区

**查看模式**：
- **标题行**："⏱ 时间安排"（H3 15px/600）
- **内容**：
  - 开始日期：📅 图标 + 日期文字（如"2025年3月1日"）+ 星期（如"周六"）
  - 结束日期：📅 图标 + 日期文字（如有）
  - 具体时间：🕐 图标 + 时间范围（如"14:00 - 16:00"，如有）
  - 提醒设置：🔔 图标 + 提醒时间（如"截止前1天提醒"或"3月14日 09:00提醒"）
  - 全天标记：如果是全天任务，显示"全天任务"标签

**编辑模式**：
- 开始日期 → 日期选择器（点击展开日历弹窗）
- 结束日期 → 日期选择器（可选，留空表示无结束日期）
- 时间范围 → 两个时间选择器（开始/结束，24小时制，30分钟步进）
- 全天开关 → Toggle 开关（开启后时间选择器隐藏）
- 提醒设置 → 下拉选择：不提醒 / 截止前1天 / 截止前3天 / 截止前1周 / 自定义时间

### 附件区

**查看模式**：
- **标题行**："📎 附件"（H3 15px/600）+ 附件数量徽章
- **附件列表**：每个附件一行，高度 48px，hover 背景 Surface Hover
  - 文件类型图标（根据类型：📄 PDF / 📝 Word / 📊 Excel / 🔗 链接 / 📧 邮件）
  - 文件名（14px/Text Primary，单行截断）
  - 文件大小（Caption/Text Tertiary，如"2.4MB"）
  - 右侧操作（hover显示）：下载图标 / 删除图标
  - 链接类型：显示 URL 域名（如"mckinsey.com"），点击在新标签页打开
  - 邮件类型：显示邮件主题+发件人，点击预览邮件内容
- **空状态**："暂无附件"（Caption/Text Tertiary）+ "添加附件"按钮

**编辑模式**：
- 附件列表上方显示"拖拽文件到此处或点击上传"区域
  - 虚线边框（2px dashed `#E8EBF0`），圆角 md，height 80px，居中
  - 上传图标 + "点击或拖拽上传"文字
  - hover：边框变为 Primary Blue，背景 Primary Light
- 每个附件右侧显示删除图标（始终可见）
- **添加链接/邮件**：
  - "添加链接"按钮 → 弹出输入框（URL + 链接名称）
  - "添加邮件"按钮 → 弹出输入框（邮件主题 + 发件人 + 内容摘要）

### 笔记区

**查看模式**：
- **标题行**："📝 笔记"（H3 15px/600）+ 笔记数量 + "添加笔记"按钮（Primary Blue边框，小尺寸）
- **笔记列表**：按时间倒序，每条笔记一个卡片
  - **笔记卡片**：
    - 顶部：笔记类型标签（Text/面试复盘/跟进记录/自定义）+ 创建时间（Caption/Text Tertiary）
    - 内容：正文（14px/Text Primary，支持换行，最多显示4行，超出"展开"按钮）
    - 底部操作（hover显示）：编辑 / 删除
    - 背景：`#FAFBFC`，圆角 md，padding 12px，margin-bottom 8px
  - **面试复盘类型**：特殊样式，左侧 3px Info Cyan 竖条，顶部显示"面试复盘"标签（Info Cyan徽章）
  - **跟进记录类型**：左侧 3px Success Green 竖条

**编辑模式**：
- 点击"添加笔记" → 展开富文本输入区域
  - 类型选择（单选标签：文本/面试复盘/跟进记录）
  - 文本域（min-height 100px，placeholder "记录你的想法、面试反馈、后续行动计划..."）
  - 底部："保存"（Primary Blue填充）+ "取消"
- 点击已有笔记的"编辑" → 该笔记卡片变为编辑态（同新增输入区域）

## 底部操作栏

固定在抽屉底部，高度 64px，padding 12px 24px，背景 `#FFFFFF`，顶部边框 1px `#E8EBF0`，阴影 `0 -4px 12px rgba(0,0,0,0.04)`。

**查看模式**：
- 左侧："编辑"按钮（Primary Blue边框，大尺寸）
- 右侧："删除"按钮（Danger Red文字按钮，图标+文字）

**编辑模式**：
- 左侧："取消"按钮（灰色边框）
- 右侧："保存"按钮（Primary Blue填充，大尺寸）+ "保存并继续编辑"（小号文字链接）

## 备注

### 双模式编辑的设计决策

任务详情采用"查看/编辑"双模式而非始终可编辑，原因如下：
1. **减少误触**：用户在查看详情时不会意外修改数据
2. **信息层级清晰**：查看模式下只展示有价值的信息，不被输入框干扰
3. **符合用户心智**：先阅读，再决定是否编辑
4. **移动端友好**：避免了触摸时意外触发编辑

### 笔记类型的设计意图

预定义三种笔记类型而非纯文本，是为了引导用户做有价值的记录：
- **文本**：通用备忘（如"记得更新简历中的项目经历"）
- **面试复盘**：结构化面试后反思（如"HR问了职业规划，回答得不够好，下次要准备2分钟的版本"），这类笔记积累下来就是面试题库
- **跟进记录**：与HR/面试官的沟通记录（如"3月5日发邮件询问进度，对方回复说下周给结果"）

用户还可以创建自定义类型（如"笔试经验"、"内推信息"），自定义类型可选择颜色。

### 附件管理的文件类型策略

支持的附件类型和图标映射：

| 文件类型 | 扩展名 | 图标 | 预览方式 |
|----------|--------|------|----------|
| PDF | .pdf | 📄 红色 | 内置PDF预览器 |
| Word | .doc, .docx | 📝 蓝色 | 下载后用本地应用打开 |
| Excel | .xls, .xlsx | 📊 绿色 | 下载后用本地应用打开 |
| 图片 | .png, .jpg, .jpeg | 🖼 紫色 | 内置图片预览（灯箱） |
| 链接 | URL | 🔗 灰色 | 新标签页打开 |
| 邮件 | - | 📧 橙色 | 内置邮件内容预览 |

### 链接附件的特殊处理

当用户添加链接附件（如职位详情页URL）时：
- 系统自动提取网页标题作为链接名称（如果用户未自定义）
- 显示网站 favicon（如果可获取）
- 点击链接时，先显示一个确认弹窗（"将在新标签页打开 external-site.com"），防止意外跳转丢失当前工作上下文

### 抽屉内嵌套弹窗的处理

抽屉打开后，其内部的操作（如删除确认、添加链接弹窗）以**覆盖在抽屉上方**的模态框形式出现，而非覆盖全屏。模态框居中于抽屉内，宽度 360px，背景遮罩仅覆盖抽屉内容区（rgba(255,255,255,0.8)）。

### 表单验证与错误提示

编辑模式下的验证规则：
- **公司名**：必填，最大50字符，为空时输入框边框变为 Danger Red + 下方显示"请输入公司名称"
- **岗位名**：必填，最大50字符
- **结束日期**：不能早于开始日期，违反时两个日期选择器同时变红 + 提示"结束日期不能早于开始日期"
- **附件大小**：单个文件最大 20MB，超限上传时 Toast 提示"文件大小超过20MB限制"

### 键盘快捷键（抽屉内）

| 快捷键 | 操作 |
|--------|------|
| `Esc` | 关闭抽屉（查看模式）/ 取消编辑（编辑模式） |
| `Ctrl/Cmd + S` | 保存（编辑模式下） |
| `Ctrl/Cmd + Enter` | 保存笔记（笔记编辑时） |
| `Ctrl/Cmd + Shift + D` | 删除任务（弹出确认） |
